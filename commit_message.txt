feat 고급 UI 및 안정적인 스탯 시스템 구현

이번 커밋은 UI 및 스탯 시스템을 대대적으로 개편하여 게임 플레이 피드백을 개선하고 치명적인 버그를 수정했습니다

UI 개선 사항

NPCUI 추가 NPC의 이름과 체력을 화면에 표시하여 전투 상황 인지도를 높였습니다
PlayerStatUI 구현 플레이어의 상세 스탯좌표 체력 공격력 속도 힘 민첩 스태미나을 화면 우측에 표시합니다
좌표 표시 리팩토링 기존 좌표 표시 로직을 새로운 UI 시스템에 통합하고 중복 코드를 제거했습니다

스탯 및 버프 시스템 개편

UpdateDerivedStats 함수 playerjs에 새로운 함수를 추가하여 기본 스탯힘 민첩이 파생 스탯공격력 속도 등에 올바르게 영향을 주도록 수정했습니다
버그 수정 버프 버프 아이템을 섭취해도 플레이어의 공격력이 업데이트되지 않던 치명적인 버그를 수정했습니다 이제 스탯이 변경되거나 무기를 장착할 때마다 UpdateDerivedStats 함수가 호출되어 즉시 업데이트됩니다

리팩토링 및 버그 수정

코드 정리 playerjs에서 사용하지 않는 attackDuration 속성을 제거했습니다
TypeError 수정 mainjs에서 NPC 모델이 완전히 로드되기 전에 접근하여 발생하던 TypeError를 null 체크를 추가하여 해결했습니다
헤드샷 렌더링 개선 hpjs에서 프로필 이미지 렌더링 시 카메라 위치를 미세 조정하여 더 나은 클로즈업 샷을 캡처하고 코드 가독성을 위해 주석을 추가했습니다
NPC 이름 지정 objectjs의 NPC 생성자에 name 파라미터를 추가하여 더 다양한 NPC를 생성할 수 있도록 했습니다


feat Implement advanced UI and robust stat system

This commit introduces a major overhaul of the UI and stat systems enhancing gameplay feedback and fixing critical bugs

UI Enhancements

NPCUI Added Displays NPC name and health onscreen improving combat awareness
PlayerStatUI Implemented Shows detailed player stats coordinates health attack speed strength agility stamina on the right side of the screen
Refactored Coordinate Display Integrated the old coordinate display logic into the new UI system removing redundant code

Stat and Buff System Overhaul

UpdateDerivedStats Function Created a new function in playerjs to properly calculate derived stats like attack damage and speed from base stats strength agility
Bug Fix Buffs Fixed a critical bug where consuming a buff item did not update the players attack damage The new UpdateDerivedStats function is now called whenever stats change or weapons are equipped ensuring immediate updates

Refactoring and Bug Fixes

Code Cleanup Removed the unused attackDuration property from playerjs
TypeError Fix Resolved a TypeError in mainjs that occurred when accessing the NPC model before it was fully loaded by adding a null check
Improved Headshot Rendering Finetuned the camera position for profile image rendering in hpjs to capture better closeup shots and added comments for clarity
NPC Naming Added a name parameter to the NPC constructor in objectjs to allow for more varied NPC creation
